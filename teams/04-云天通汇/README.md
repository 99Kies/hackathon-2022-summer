## 项目整体简介

创建一个新的数据处理器的实例，类似于现有的 Substrate Frontier EVM 处理器。
它要连接到 Gear Wasm 合约平台（最好也能在其他 Wasm 合约平台上工作），
并以类似于处理来自 Substrate Frontier EVM 处理器的消息的方式，
处理来自 Wasm 执行的消息。
然后，你可以在处理程序中使用这些信息，就像传统的 SubQuery 项目一样。
